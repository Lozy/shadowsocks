shadowsocks
===========

```
wget --no-check-certificate https://raw.githubusercontent.com/Lozy/shadowsocks/master/SS_debian -O SS_debian
chmod +x SS_debian
./SS_debian
```

|  --password   |       shadowsocks password[Random] |
|  --encry      |       package encry type |
